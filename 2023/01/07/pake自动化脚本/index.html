基于rust打包网页应用

本文使用开源项目工具Pake：tw93/Pake: 🤱🏻 Simply make any web page a desktop application using Rust. 🤱🏻 很简单的用 Rust 打包网页生成很小的桌面 App (github.com)

在使用前需要先安装VS相关tool以及rust，随后使用npm安装pake包，教程参见上述链接

下表给出使用命令行安装的常用命令：

在本人使用过程中发现必须在nodejs/node_global/node_modules/pake-cli目录下以管理员方式打开才可以使用，十分不便，因此编写批处理程序简化操作。

在bat中获取管理员权限有多种方法，代码参考链接：bat以管理员运行

url 打包的网页链接,必须提供 pake < url >
name 应用名称 --name < value >
icon 应用图标，支持本地/远程文件,Windows必须为.ico,MacOS必须为.icns,Linux必须为.png --icon < path >

set uac=~uac_permission_tmp_%random%
md "%SystemRoot%\system32\%uac%" 2>nul
if %errorlevel%==0 ( rd "%SystemRoot%\system32\%uac%" >nul 2>nul ) else (
    echo set uac = CreateObject^("Shell.Application"^)>"%temp%\%uac%.vbs"
    echo uac.ShellExecute "%~s0","","","runas",1 >>"%temp%\%uac%.vbs"
    echo WScript.Quit >>"%temp%\%uac%.vbs"
    "%temp%\%uac%.vbs" /f
    del /f /q "%temp%\%uac%.vbs" & exit )
endlocal

随后给出需要的三个参数即可
