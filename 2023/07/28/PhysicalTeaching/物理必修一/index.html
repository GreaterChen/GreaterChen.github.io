高中物理必修一

运动的描述

物体和质点

  1. 质点： 忽略物体的大小和形状，把它简化为具有质量的点，叫做质点
  2. 区分：具体情况具体分析，一个物体可以看作质点，也可以不看做质点

静止、绝对静止、相对静止

  1. 静止：通常是相对于地面
  2. 绝对静止：绝对静止是不存在的
  3. 相对静止：主要看参考系的选择：以地面为参考系，房子是静止的；但是以月球为参考系，房子是运动的。

参考系

  1. 定义:用来参考的物体
  2. 应用：非惯性参考系
  3. 注意点：不同物体的运动情况或者同一物体在不同阶段的运动情况时，必须选择同一个参考系

时刻、时间间隔

  1. 区分：时刻是一个点，时间间隔是一个区间
  2. 第一秒初、第一秒末

路程和位移

  1. 路程：物体的运动轨迹的长度

  2. 位移：初位置指向末位置的有向线段

下列关于位移和路程的说法正确的是 （CD） A.物体沿直线向某一方向运动，通过的路程就是位移
B.物体沿直线运动，通过的路程等于位移的大小 C.物体通过的路程不相等，位移可能相同 D.物体通过一段路程，位移可能为零

位移-时间图像(x-t图像)

  1. 横坐标：时间
  2. 纵坐标：位移
  3. 斜率：速度
  4. 截距：初始值所处的位置
  5. 注意点：x-t图像只能表示在直线上的运动！曲线表示的是变速直线运动

速度、瞬时速度、平均速度

  1. 速度的定义：\(v = \frac{\Delta x}{\Delta t}\), \(x\)是位移

  2. 瞬时速度：\(\Delta t\)趋于0的速度

  3. 平均速度：任意一个时间间隔内运动的平均快慢程度

  4. 速率：瞬时速度的大小

  5. 平均速率：路程和时间的的比值\(\frac{\Delta s}{\Delta t}\)

  6. 平均速度和平均速率什么时候相同？

    位移大小和路程相同的时候：单向直线运动时

速度-时间图像(v-t图像)

  1. 横轴：时间

  2. 纵轴：速度

  3. 斜率：加速度

  4. 截距：初速度

  5. 和x轴围成的面积：位移

    无论v-t图像多不规则，围成的面积都代表位移：微分的思想

加速度

  1. 定义：速度的变化量与发生这一变化所用的时间的比值

    \(a = \frac{\Delta v}{\Delta t}\)

  2. 速度和加速度的关系

    1. 加速度大小和速度大小没有必然关系
    2. 速度为0的时候，加速度不一定为0；加速度为0的时候速度也不一定为0
    3. 加速度增大速度不一定增大；加速度减小速度也不一定减小
    4. 速度增大加速度不一定增大；速度减小加速度也不一定减小
    5. 加速度增大，速度的变化程度一定会变大 \(\Delta v = a*\Delta t\)

匀变速直线运动的研究

匀变速直线运动

  1. 定义：沿着一条直线，且加速度不变的运动
  2. 速度与时间的关系：\(v = v_0 + a*t\)
  3. 速度和加速度是否成正比？ 不成正比；呈线性变化
  4. 在v-t图像中，是一条倾斜的直线

AD

位移与时间的关系

  1. 公式：\(x = v_0t+\frac{1}{2}at^2\)

  2. 求导：\(\Delta x /\Delta t\) = \(v_0+at\)

BD

  1. 当x的式子有常数时，常数代表初始的位置

  2. 求导：常数直接消失；对于\(t^n\)的，求导之后都是\(nt^{n-1}\); \(t^2\)->\(2t\); t->1

  3. v = 5 - 2t ; a = -2

ABD

  1. \(x=1/2at^2\),初速度为0，加速度为2

  2. \(\Delta x = aT^2\)

  3. 不好算的画v-t图像看比较直观

速度与位移的关系

  1. 公式 \(v^2 - v_0^2 = 2ax\)

  2. \(x = \frac{1}{2a}v^2 - \frac{v_o^2}{2a}\)

  3. 在匀变速直线运动中，\(\bar v = \frac{v_1+v_2}{2}\)

初速度为0的匀加速直线运动的常用推论

  1. 等分时间
    1. T末、2T末、3T末...nT末，瞬时速度之比为：1：2：3:...:n （v=at，速度与t成正比）
    2. 前T内、前2T内、前3T内....前nT内，位移之比：1：4：9：....:\(n^2\)（x=\(1/2at^2\),位移与\(t^2\)成正比）
    3. 第一个T内、第二个T内。。。第n个T内，位移之比：1：3：5：....:(2n-1)
  2. 等分位移
    1. 通过x,2x,3x,...,nx所用的时间之比：\(1:\sqrt{2}:...:\sqrt{n}\)
    2. 通过第一个x，第二个x...第n个x的时间之比：\(1:\sqrt{2}-1:...:\sqrt{n}-\sqrt{n-1}\)
    3. x末、2x末...nx末的瞬时速度之比：\(1:\sqrt{2}:...:\sqrt{n}\)

刹车问题是反向的初速度为0的匀加速直线运动

自由落体

  1. 特点：初速度为0，加速度为g
  2. v = gt
  3. h = \(\frac{1}{2}gt^2\)
  4. 末速度\(v = \sqrt{2gh}\)
  5. 下落时间\(t = \sqrt{\frac{2h}{g}}\)

相互作用——力

力

  1. 作用效果：1. 使物体发生形变；2. 改变物体的运动状态
  2. 力的三要素：大小、方向、作用点

重力

  1. 施力物体是地球
  2. 大小：G=mg
  3. 方向：始终是竖直向下的
  4. 重力在大多数情况下只是地球对物体吸引力的一个分力
  5. 确定重心：1. 质量分布均匀、形状规则的，在几何中心；2. 形状不规则的，用悬挂法确定重心位置

弹力

  1. 形变：弹性形变和非弹性形变
  2. 产生条件：接触且物体发生形变
  3. 方向：与物体恢复形变的方向相同
  4. 大小：F=kx，k是劲度系数
  5. 轻绳、轻弹簧力的突变
    1. 剪断轻绳，绳的弹力会突变，物体只受重力
    2. 简短弹簧上面的轻绳，弹簧的弹力保持不变，物体还是受重力和弹力，维持平衡
  6. 弹簧的串并联
    1. 弹簧的串联：\(\frac{1}{k} = \frac{1}{k_1} + \frac{1}{k_2} + ... + \frac{1}{k_n}\)(和电阻的并联相同)
    2. 弹簧的并联：\(k = k_1 + k_2 +...+k_n\)(和电阻的串联相同)

摩擦力

  1. 滑动摩擦力
    1. 产生条件
      1. 接触并挤压
      2. 两个物体间有相对运动
      3. 接触面粗糙
    2. 方向：跟接触面相切，并且跟相对运动的方向相反
    3. 大小：\(f=\mu F_N\)
    4. 作用效果：阻碍两个物体之间的相对运动
  2. 静摩擦力
    1. 产生条件
      1. 接触并挤压
      2. 两物体有相对运动趋势
      3. 接触面粗糙
    2. 方向：跟接触面相切，和相对运动趋势相反
    3. 大小：0~\(F_{max}\),不能用滑动摩擦力的公式计算，只能通过受力分析进行计算

牛顿第三定律

  1. 内容：两个物体之间的作用力和反作用力总是大小相等，方向相反，作用在同一条直线上的。

题

运动和力

牛顿第一定律

  1. 内容：一切物体总保持匀速直线运动状态或静止状态，除非作用在它上面的力迫使他改变这种状态
  2. 力并不是维持物体运动的原因，而是改变物体运动的原因

惯性

  1. 定义：物体这种保持原来匀速直线运动或者静止状态的性质叫做惯性
  2. 量度：只和质量有关系

牛顿第二定律

  1. 物体加速度的大小和它受到的作用力成正比，跟它质量成反比，加速度的方向和作用力的方向相同
  2. 公式:F=ma

超重和失重

  1. 超重
    1. 物体对支持物的压力大于物体所受重力
    2. 本质：具有向上的加速度
  2. 失重
    1. 物体对支持物的压力小于物体所受重力
    2. 本质：具有向下的加速度、
    3. 完全失重：物体与支持物的压力为0
