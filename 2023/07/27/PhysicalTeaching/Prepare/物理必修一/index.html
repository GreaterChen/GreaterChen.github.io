高中物理必修一

运动的描述

质点、参考系

物体和质点

  1. 质点：在某些情况下，忽略物体的大小和形状，把他简化为一个具有质量的点，这个点称之为质点

  2. 如何区分：具体情况具体分析：面对面打招呼是物体，在50层楼看行人是质点（忽略了个体属性，只关注其存在）

  3. 高中物理主要研究的是质点

参考系

  1. 静止、绝对静止、相对静止

    1. 绝对静止：自然界一切物体都处于永恒的运动中，绝对静止的物体是不存在的
    2. 静止：通常来说是相较于地面的
    3. 相对静止：主要看参考系的选取

  2. 参考系

    1. 定义：用来作为参考的物体

    2. 举例：我站在地面上看列车上的某人A是运动的，列车里面的人B看A是静止的。

      两个木块，给下面木块一个力，不同参考系下上面木块的速度方向

      解答：取河水为参考系，则相对于河流，船顺流和逆流的速度是一样的。救生圈在落水之后是静止的。B->P一小时能到则P->B也是一个小时，则A->P为五个小时，故为11点。

    3. 应用：非惯性参考系——合理选择参考系可以简化问题

    4. 注意点：比较不同物体的运动情况或同一物体在不同阶段的运动情况时，必须选择同一个参考系

时间、位移

时刻、时间间隔

  1. 区分：时刻是一个点，时间间隔是一个区间

路程和位移

  1. 路程：物体运动轨迹的长度

  2. 位移：初位置指向末位置的有向线段

    涉及到运动的叠加：正方向移动的距离和负方向移动的距离不能单纯相加

  3. 下列关于位移和路程的说法正确的是 （） A.物体沿直线向某一方向运动，通过的路程就 是位移 B.物体沿直线运动，通过的路程等于位移的大小 C.物体通过的路程不相等，位移可能相同 D.物体通过一段路程，位移可能为零

位移-时间图像(\(x-t\)图像)

  1. 横坐标：时间

  2. 纵坐标：位移

  3. 斜率：速度

  4. 截距：初始时的位置

  5. 只能表示在直线上的运动！曲线代表做变速直线运动

## 速度

速度、瞬时速度、平均速度

  1. 速度定义：\(v=\frac{\Delta x}{\Delta t}\)
  2. 瞬时速度：\(\Delta t\)->0时的速度
  3. 平均速度：在任意一个时间段\(\Delta t\)内运动的平均快慢程度
  4. 速率：瞬时速度的大小
  5. 平均速率：路程与时间的比值(\(\frac{\Delta s}{\Delta t}\))
  6. 平均速度和平均速率什么时候一样？做单向直线运动时

速度-时间图像(\(v-t\)图像)

  1. 横轴：时间

  2. 纵轴：速度

  3. 斜率：加速度

  4. 截距：初始速度

  5. 围成的面积：位移

    无论v-t图像有多不规则，围城的面积都代表位移

加速度

  1. 定义：速度的变化量与发生这一变化所用时间之比

    \[a = \frac{\Delta v}{\Delta t}\]

  2. 速度和加速度的关系

    1. 加速度大小和速度的大小没有必然关系

    2. 速度为0的时候，加速度不一定为0；加速度为0时速度也不一定为0

    3. 加速度增大速度不一定增大；加速度减小速度也不一定减小

    4. 速度增大加速度不一定增大，速度剑侠加速度也不一定减小

    5. 加速度变大，速度的变化趋势会变大
