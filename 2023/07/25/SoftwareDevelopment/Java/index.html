Java基础学习

代码块

  1. 静态代码块:
    1. 格式：static { }
    2. 特点：类加载时自动执行，由于类只会加载一次，所以静态代码块也只会执行一次。
    3. 作用：完成类的初始化，例如：对类变量的初始化赋值。
  2. 实例代码块
    1. 格式：{ }
    2. 特点：每次创建对象时，执行实例代码块，并在构造器前执行。
    3. 作用：和构造器一样，都是用来完成对象的初始化的，例如：对实例变量进行初始化赋值。

单例设计模式

  1. 保证一个类只有一个对象，防止内存占用
  2. 定义方法：把类的构造器私有；定义一个类变量存储类的一个对象；提供一个类方法返回对象
