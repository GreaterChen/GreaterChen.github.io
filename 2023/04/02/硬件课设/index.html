硬件课设

本文最后更新于：2023年5月9日 下午

记录硬件课设学习过程

实验一 PL端设计

  1. 新建工程，编写好.v文件
  2. Synthesis -> Run Synthesis 检查是否具有语法错误
  3. 新建simulation sources，编写仿真文件
  4. 将仿真文件右键设置为top文件，Simulation -> Run Simulation 进行仿真
  5. I/O Ports 设置引脚
  6. Program and Debug -> Generate Bitstream 产生字节流
  7. Open Hardware Manager 将程序烧到电路板

实验二 PS端设计

  1. 新建工程，IP Integrator -> Create Block Design

  2. 查找zynq，选择第一个

  3. 双击芯片，添加预配置文件

  4. MIO Configuration -> I/O Peripherals -> GPIO -> EMIO GPIO (Width) 打勾，选择引脚数量

  5. 将GPIO_0 DDR FIXED_IO三个引脚右键Make External引出

  6. 点击Validate Design 审查是否具有错误

  7. 右键IP文件选择Generate Output Product, 点击Generate

  8. 右键IP文件点击Create HDL Wrapper进行封装

  9. Run Synthesis

  10. Add Source -> Add or create constraints 新建约束文件并填写

  11. Generate Bitstream

  12. File->Export->Export Hardware->Include bitstream

  13. Launch SDK

  14. 在SDK中File->New->Application Project->Next->empty Project

  15. bsp文件夹/system.mss中有一些测试示例，选择gpiops的Import Examples

  16. 修改文件逻辑

  17. 在Vivado中Open Target 连接电路板

  18. 在SDK中Run as -> Launch on Hardware(System Debugger) 烧到电路板上

实验三 PS + PL已有IP

  1. 同实验二的1~3，然后打开AXI总线并在Interrupts勾选Fabric Interrupts,展开后在PL-PS Interrupt Ports勾选IRQ_F2P[15:0]开启中断

  2. 添加另一个IP查找GPIO，添加后双击勾选Enable Dual Channel开启双使能通道

  3. 对于两个GPIO宽度都设置为4(LED和按键都是占4个引脚)

  4. 勾选Enable Interrupt，点击确定

  5. 引出DDR FIXED_IO GPIO DPIO2

  6. 点击页面上方的自动连接将IP和ZYNQ连接起来，然后手动将ZYNQ和IP的中断接口连起来

  7. 同实验二6~12

  8. File->new->Board Support Package 全默认即可

  9. system.mss中勾选gpio的前两个示例代码

  10. 同实验二后续

实验四 PS + PL自定义IP

  1. 新建工程编准备好.v文件

  2. Tools->Create and Package New IP->Create a new AXI4 peripheral->Edit IP

  3. 添加资源，将需要封装为IP的文件（记作A）引入进来

  4. 将文件中的module声明复制到自动产生文件（记作B）的末尾 user logic处，同时指定端口的连接

  5. 在Users to add ports here处添加IP的输入输出，注意reg要改为wire

  1. 在top层两处添加端口

  1. Run Synthesis
  2. 在Package IP - AXI_SMG->File Groups页面点击Merge changes from File Grops Wizard,在下面一个页面进行同样的操作
  3. 点击最下面Review and Package完成IP打包
  4. 新建项目，IP Integrator->Create Block Design打开IP设计界面
  5. 在Project->Project Setting中的IP页面中添加IP的路径
  6. 预配值Zynq，勾选AXI总线（默认已勾选）
  7. 引出DDR FIXED_IO led_out a_g key_in四个引脚，然后点击自动连接
  8. 同实验三后续

正点原子

  1. 嵌入式开发流程

  1. GPIO是一个外设，用来对器件的引脚作观测(input)以及控制(output)。
  2. MIO(Multiuse I/O),将来自PS外设和静态存储器接口的访问多路复用到PS的引脚上。

OLED

    2. SCLK:D15

    3. CS:C15

    4. GPIO[0]:D16

    5. GPIO[1]:F17

  3. ss: T18
